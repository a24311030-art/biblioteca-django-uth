# 📚 Sistema de Biblioteca - Django & MySQL

## 📋 Descripción
Sistema de gestión de biblioteca desarrollado con Django y MySQL como práctica de la Unidad 1.

## 🚀 Tecnologías Utilizadas
- Python 3.x
- Django 4.x
- MySQL 8.x
- HTML5, CSS3, Bootstrap 5
- Git & GitHub

## ⚙️ Instalación
1. Clonar repositorio: \git clone https://github.com/TU-USUARIO/biblioteca-django-uth.git\
2. Crear entorno virtual: \python -m venv venv\
3. Activar entorno: \env\Scripts\activate\ (Windows)
4. Instalar dependencias: \pip install -r requirements.txt\
5. Configurar base de datos MySQL
6. Ejecutar: \python manage.py runserver\
